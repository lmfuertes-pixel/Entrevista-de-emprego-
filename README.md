# Simulação de Entrevista de Emprego

Este projeto faz parte da entrega na plataforma DIO.  
O objetivo é simular uma entrevista de emprego e registrar perguntas e respostas, com comentários adicionais.

---

## Perguntas e Respostas

1. **Pode me contar um pouco sobre você e sua trajetória profissional até aqui?**  
   - **Resposta:** Técnico em manutenção de computadores e instalação de software migrando para carreira de IA.  
   - **Comentário:** Mostra experiência prática e transição clara para uma área de inovação.

2. **O que te motivou a escolher a área de tecnologia e, mais recentemente, dar esse passo para a Inteligência Artificial?**  
   - **Resposta:** A facilidade que tenho em me adaptar.  
   - **Comentário:** Adaptabilidade é uma competência muito valorizada em ambientes dinâmicos.

3. **Qual projeto ou experiência você considera mais marcante na sua carreira ou nos seus estudos até agora?**  
   - **Resposta:** Projetos de instalação de grandes quantidades de hardware (lan house, escritório de contabilidade, automação). Nos estudos, ser assertivo em 99% dos questionários.  
   - **Comentário:** Demonstra tanto experiência prática quanto disciplina acadêmica.

4. **Quais linguagens de programação você domina ou tem mais afinidade?**  
   - **Resposta:** Nenhuma.  
   - **Comentário:** Honestidade é importante; mostra que está em fase inicial de aprendizado em programação.

5. **Você já trabalhou com versionamento de código (Git/GitHub)? Como costuma organizar seus commits?**  
   - **Resposta:** Já trabalhei com GitHub para entregar projetos anteriores do curso de IA. Organizo commits de forma resumida e em inglês quando necessário.  
   - **Comentário:** Boa prática, usar inglês e manter commits claros.

6. **Se tivesse que explicar para alguém leigo o que é uma API, como você faria?**  
   - **Resposta:** API são padrões usados para permitir intercâmbio de dados e ações entre softwares.  
   - **Comentário:** Explicação simples e direta, adequada para leigos.

7. **Imagine que você precisa corrigir um bug urgente em produção. Qual seria seu passo a passo?**  
   - **Resposta:** Usar um backup de uma versão estável para resolver o problema imediato e depois procurar uma solução definitiva.  
   - **Comentário:** Mostra pensamento em contingência e solução definitiva.

8. **Como você lida com prazos apertados e pressão no trabalho?**  
   - **Resposta:** Com tranquilidade.  
   - **Comentário:** Demonstra equilíbrio emocional.

9. **Você prefere trabalhar sozinho ou em equipe? Por quê?**  
   - **Resposta:** Sozinho, pois tenho mais controle das minhas funções. Em grupo quando há um projeto de grandes dimensões, pois é inevitável.  
   - **Comentário:** Mostra consciência sobre quando o trabalho em equipe é necessário.
